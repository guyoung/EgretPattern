GyEgretPattern
==============
Egret应用开发实践

## 使用

### 安装依赖包 

    npm install

### 编译应用

在应用目录下，如apps/app_01_hello_world，执行

     ..\..\node_modules\.bin\webpack

### 运行

在项目目录下，执行

    npm run start
